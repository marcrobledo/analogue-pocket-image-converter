# Analogue Pocket Image Converter
This tool can convert between images and Analogue Pocket image formats.<br/>
It can be used to customize your Analogue Pocket OS look and feel by changing openFPGA core banners and adding game thumbnails to the library grid view.

[See it in action here](https://www.marcrobledo.com/analogue-pocket-image-converter)

**Supported formats:**
- developer icons (36x36, `/Cores/*/icon.bin`)
- core banners (521x65, `/Platforms/_images/*.bin`)
- library screenshots (`/System/Library/Images/*/*.bin`)
- library grid thumbnails (`/System/Library/Images/*_thumbs.bin`)
- library grid list (`/System/Played Games/list.bin`)
